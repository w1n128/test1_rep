# Свалкус

Маленькая 2D-игра в стиле Том и Джерри: дворник пытается поймать енота при помощи мышеловок, петард, люков, банановых кожурок и предметов для драки. Енот — то же самое в обратную сторону. Подбираются припасы, есть скрытность, ночной режим и ловушки.

Сделано на чистом HTML5 Canvas + WebAudio + WebRTC, без бэкенда.

## Быстрый старт (локально)

В терминале из этой папки:

```bash
python3 -m http.server 8765
```

Затем открыть в браузере: `http://localhost:8765/`

## Режимы игры

1. **1P vs CPU** — один игрок против AI-енота. WASD движение, F использовать выбранный предмет, Q/E переключить предмет.
2. **2P Hot Seat** — два игрока за одной клавиатурой. Дворник: WASD/F/Q/E. Енот: стрелки, `.`, `/`, `,`, Right Shift.
3. **По сети** — игра с другом через интернет (см. ниже).
4. **Режим новичка** — упрощённый 1P vs CPU: меньше предметов, нет ночи и случайных событий, скорость ниже на 30%.

## Игра по сети

Архитектура: WebRTC P2P через PeerJS-broker. Один игрок становится **хостом**, второй — **клиентом**. Никакой регистрации — всё по короткому 6-символьному коду комнаты.

### Шаги

1. **Хост**: открыть страницу, выбрать `3. По сети` → `Создать комнату`. На экране появится код, например `K7P2X9`.
2. **Скинуть код другу** любым способом (Telegram, мессенджер).
3. **Друг (клиент)**: открыть ту же страницу, выбрать `3. По сети` → `Войти в комнату` → набрать код на клавиатуре → Enter.
4. Через 1-2 секунды установится прямое P2P-соединение, игра стартует.
   - Хост управляет **Дворником** (WASD/F/Q/E).
   - Клиент управляет **Енотом** (тоже WASD/F/Q/E на своей клавиатуре).
5. Каждый видит **только свою камеру** на полный экран.

### Если не подключается

- Подождать 10 сек: первичная регистрация в broker иногда тормозит.
- Перезапустить обе вкладки (Esc → Esc → 3 → ...).
- При симметричном NAT (редко, но бывает у мобильных операторов) — попробуйте включить TURN-сервер; в `src/net.js` уже прописан публичный openrelay TURN как fallback.

## Управление

| Действие | Игрок 1 (Дворник) | Игрок 2 (Енот, hot-seat) |
|---|---|---|
| Движение | W A S D | стрелки |
| Рывок | Left Shift | Right Shift |
| Использовать выбранный предмет | F | `.` |
| Следующая ловушка | E | `/` |
| Предыдущая ловушка | Q | `,` |

В сетевом режиме каждый клиент использует своё «Управление 1» (WASD/Shift/F/Q/E).

Перед стартом матча показывается краткая памятка правил с 5-секундным обратным отсчётом.

| Системные | |
|---|---|
| Esc | Пауза (показывает правила и описание ловушек) |
| M | Назад в меню |
| R | Рестарт матча на экране победы |
| 1 / 2 / 3 / 4 | Быстрый выбор режима из меню |

## Игровые предметы

- **Мышеловка** — наступание = 1 урон, ломается. До 3 в инвентаре.
- **Петарда** — фитиль 0.735 сек, взрыв радиусом 3 тайла, урон 1. До 3.
- **Люк** — невидим противнику, проваливает на 2 сек. До 3.
- **Банановая кожура** — броcок на 3 тайла вперёд, скольжение мгновенно. До 3.
- **Ветка** — используется по F: одноразовый удар на 1 клетку перед собой, урон 1 + микростан.
- **Банка** — бросается по направлению взгляда на 5 клеток, попадание даёт 1 урон + короткую остановку.

В **режиме новичка** доступны только мышеловка, петарда, банановая кожура и ветка.

## Спецпредметы

- **Звезда** — 10 сек бессмертия и ускорения; музыка ускоряется, персонаж мигает. При успешном ударе звездой эффект сразу пропадает.
- **Метла** — делает персонажа невидимым для соперника на 5 сек и оставляет пылевой след.
- **Магнитофон** — ставится на текущую клетку, заставляет владельца танцевать и лечит по 1 HP каждые 2 сек. Повторное F выключает магнитофон.
- **Алмаз** — стартовый предмет дворника; по F притягивает енота к дворнику через всю карту.
- **Пицца** — стартовый предмет енота; по F притягивает дворника к еноту через всю карту.

## Стартовый инвентарь

- Каждый игрок начинает с 2 мышеловками, 1 петардой, 1 люком, 1 банановой кожурой, 1 веткой и 1 банкой.
- В режиме новичка стартовый инвентарь проще: 2 мышеловки, 1 петарда, 1 банан и 1 ветка.
- При 1 HP включается последний шанс: рывок сразу готов, скорость немного выше.
- Дворник дополнительно начинает с 1 алмазом.
- Енот дополнительно начинает с 1 пиццей.
- На карте одновременно может быть до 10 активных ловушек каждого игрока.

## Ночь и события

- После 90 сек матча наступает ночь на 60 сек; цикл повторяется.
- Ночью экран темнеет, насыщенность снижена, а динамические объекты видны только в луче фонарика на 3 клетки вперёд.
- Случайные события двора происходят примерно раз в 32-44 сек: **ветер** сдвигает свободные предметы и бананы, **мусорный дождь** добавляет новые припасы, **катящийся бак** пересекает арену и сбивает игроков.
- В режиме новичка ночь, случайные события, рывок, звезда, метла, магнитофон, люк, банка, алмаз и пицца отключены.

## Структура проекта

```
src/
  config.js     — все балансные константы и палитра
  audio.js      — WebAudio: SFX + процедурный саундтрек
  particles.js  — частицы (искры, пыль, звёзды)
  sprites.js    — все спрайты (32×32 пиксельная графика, генерируется в браузере)
  arena.js      — раскладка препятствий
  input.js      — клавиатурный ввод, маппинг
  player.js     — класс игрока, движение, состояния
  traps.js      — менеджер ловушек, эффекты
  pickups.js    — спавн и подбор припасов
  ai.js         — AI-противник для 1P vs CPU
  hud.js        — HUD: HP, инвентарь
  render.js     — split-screen и single-camera, screen-shake
  net.js        — WebRTC P2P через PeerJS, NetInputDevice
  main.js       — state machine, меню, лобби, game loop
index.html
style.css
```
